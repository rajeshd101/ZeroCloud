# ZeroCloud User Guide

## Quick Start

1. Install ZeroCloud on devices you want to connect
2. Ensure devices are on the same network
3. Launch ZeroCloud on all devices
4. Devices will automatically discover each other
5. Select a device and start transferring files

## File Transfer

### Sending Files
- Drag and drop files onto the ZeroCloud window
- Or use the file browser to select files
- Choose the target device
- Files are automatically encrypted and sent

### Receiving Files
- Accept incoming transfer requests
- Files are saved to your Downloads folder
- All files are automatically decrypted

## Security

- All files are encrypted with AES-256
- No data leaves your local network
- New encryption keys for each session
- No cloud storage or internet connection required

## Troubleshooting

### Common Issues
- **No devices found**: Check network connection and firewall
- **Transfer failed**: Verify disk space and network stability
- **Slow transfers**: Use wired connection for large files
